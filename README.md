# Wireframe Looking Website

A wireframe-style starter template built with React, TypeScript, Tailwind CSS v4, and Vite. Uses colorful dashed borders to visually map out layout structure — useful as a starting point or reference when planning a new site.

Created as a foundation for [Unirend](https://github.com/keverw/unirend/), an open source universal renderer supporting SSR and SSG.

## Features

- Wireframe-style layout with color-coded dashed borders
- Dark mode support with a theme toggle
- Responsive header, main content area, and footer
- Built with Bun, Vite v6, Tailwind CSS v4, and TypeScript

## Getting Started

Requires [Bun](https://bun.sh/).

```bash
bun install
bun run dev
```

Other commands:

```bash
bun run build        # Production build
bun run preview      # Preview production build
bun run format       # Format code with Prettier
bun run format:check # Check formatting
bun run lint         # Lint with ESLint
```
